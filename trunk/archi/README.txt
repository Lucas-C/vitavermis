Ce dossier contient la documentation de réference de l'architecure de PPG-VitaVermis.

Il se compose de plusieurs types de documents :
- schémas de classes UML
- "use-case" scenarios
- détails des responsabilités / API / points d'implémentation de chaque module / manager
- le pseudo-code de la "main game loop"
- les "requirements" des différentes versions du jeu

Cette documentation est entièrement en français.
