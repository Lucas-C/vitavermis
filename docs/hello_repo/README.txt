Salut tous le monde !
Si vous avez recuperer ce fichier, c'est que avez reussi a importer le depot (ou "repository", abrege en "repo") correctement.

Votre mission si vous l'acceptez, est maintenant de creer, dans le dossier "media", un dossier a votre nom au format suivant : votre nom tout en majuscule, suivi de la premiere lettre de votre prenom (exemple present dans le dossier "media").

une fois le dossier cree, je veux que vous y placiez un fichier de votre choix (texte, image). 

Une fois cela fais, vous faites un commit pour que vos modification soit bien pris en compte sur le depot, en n'oubliant pas de bien mettre un petit message lors de votre commit pour expliquer ce que vous avez fait (generalement, on ecrit pas plus d'une phrase).

Je pense que ce premier exercice s'arretera la, tous le nmonde n'a pas besoin d'apprendre a gerer les conflits pour l'instant.
